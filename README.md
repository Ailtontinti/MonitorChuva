## AgroPluvi - SaaS de monitoramento inteligente de chuva

Monorepo contendo:

- **backend**: API REST em Node.js + TypeScript + PostgreSQL, multi-tenant por organização.
- **mobile**: app React Native (Expo) para produtores rurais, fazendas e cooperativas.

Funcionalidades principais:

- Registro de **organizações** (fazendas, cooperativas), **usuários**, **propriedades** e **pluviômetros**.
- Registro e acompanhamento de **volumes de chuva** (rainfall records).
- Visualização de **histórico** e **dashboards** por propriedade/pluviômetro.
- Integração preparada para **previsão climática** e **alertas automáticos** (chuva excessiva, estiagem, etc.).
